# Roast My Resume

- Upload your resume and let an AI tech recruiter ruthlessly roast your experience, skills, GitHub, and LinkedIn in seconds.

- Get detailed feedback on your resume, GitHub profile, and LinkedIn activity to improve your chances of landing that dream job.

`Tech Stack`

- Next.js 16
- TypeScript
- Tailwind CSS
- Shadcn UI
- Gemini Api
- Vercel AI Sdk
